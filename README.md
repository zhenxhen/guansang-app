# 관상 앱 (Guansang App)

얼굴 특징을 분석하고 ㄴㄴ시각화하는 웹 애플리케이션입니다. MediaPipe의 Face Landmark 기술을 활용하여 실시간으로 얼굴의 비율, 대칭성, 눈, 코, 입 등의 특징을 분석합니다.

## 주요 기능

- 실시간 얼굴 인식 및 랜드마크 추적
- 얼굴 대칭성 및 비율 분석
- 눈, 코, 입 등 세부 특징 측정
- 시각적 피드백 및 결과 표시
- 모바일 및 데스크톱 환경 지원

## 기술 스택

- React.js: 프론트엔드 UI 구현
- TypeScript: 타입 안전성과 개발 효율성 강화
- MediaPipe: 얼굴 랜드마크 인식 및 추적
- Canvas API: 시각적 효과 및 그래픽 처리
- 반응형 디자인: 다양한 화면 크기 지원

## 시작하기

### 필요 조건

- Node.js 14.x 이상
- npm 또는 yarn

### 설치 및 실행

1. 저장소 복제
   ```bash
   git clone https://github.com/zhenxhen/guansang-app.git
   cd guansang-app
   ```

2. 의존성 설치
   ```bash
   npm install
   # 또는
   yarn
   ```

3. 개발 서버 실행
   ```bash
   npm start
   # 또는
   yarn start
   ```

4. 브라우저에서 앱 열기
   ```
   http://localhost:3000
   ```

2. 배포된 웹사이트는 https://zhenxhen.github.io/guansang-app/ 에서 확인할 수 있습니다.

## 빌드 및 배포

### 프로덕션용 빌드

```bash
npm run build
# 또는
yarn build
```

### GitHub Pages 배포

이 프로젝트는 GitHub Actions를 사용하여 GitHub Pages에 자동 배포됩니다.

1. `main` 브랜치에 변경사항을 푸시하면 자동으로 배포가 트리거됩니다.
2. 배포된 웹사이트는 https://zhenxhen.github.io/guansang-app/ 에서 확인할 수 있습니다.

수동 배포를 원한다면:

```bash
npm run deploy
```

## 작동 방식

1. 웹캠을 통해 얼굴을 인식합니다.
2. MediaPipe의 Face Landmarker가 468개의 얼굴 랜드마크를 추적합니다.
3. 랜드마크 좌표를 기반으로 얼굴 특징을 계산합니다.
4. 계산된 특징을 Canvas에 시각화하고 분석 결과를 표시합니다.
5. 사용자는 실시간으로 얼굴 분석 결과를 확인할 수 있습니다.

## 주의사항
